## Overview
TripTok is a **modern travel website** designed to showcase top destinations, travel packages, and user reviews. The website allows users to explore featured destinations, search for cities or countries, view travel galleries, read user reviews, and book trips with special discounts.  

This project is built with **HTML, CSS,MediaQuery and Font Awesome icons** for a clean and interactive UI.

## Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices.
- **Featured Destinations**: Showcases top travel destinations with pricing and ratings.
- **Travel Gallery**: Interactive gallery displaying popular locations.
- **User Reviews**: Section for showcasing customer testimonials.
- **Call-to-Action Section**: Highlights current discounts and offers.
- **Footer & Contact Info**: Quick access to contact details and social media.
